Назначение
Система для логирования ML-экспериментов. Сохраняет параметры обучения, метрики и пути к артефактам в структурированную базу данных.

Стек технологий

FastAPI: Серверная часть

SQLAlchemy / PostgreSQL: База данных

Loguru: JSON-логирование

Docker: Контейнеризация

Цепочка работы
ML Скрипт → AyimApi Клиент → FastAPI Роутер → Сервис → Репозиторий → PostgreSQL

Инфраструктура

Docker Compose: Запуск всей системы одной командой.

Авто-миграции: SQL-скрипты применяются автоматически при старте.

Healthchecks: API ждет готовности базы данных.

Связи данных

Эксперимент (1) : (N) Запусков — У одного проекта может быть много попыток обучения.

Запуск (1) : (N) Параметров/Метрик — Каждая попытка хранит свои уникальные данные.

Пример использования (JSON)

Создать Эксперимент (POST /experiments):

JSON
{
  "name": "Spam_Filter",
  "description": "Обучение модели на базе BERT"
}
Создать Запуск (POST /runs):

JSON
{
  "experiment_id": 1,
  "version": 1,
  "status": "running",
  "parameters": [{"key": "batch_size", "value": "32"}],
  "metrics": [{"key": "f1_score", "value": 0.88}],
  "artifacts": [{"file_path": "s3://bucket/model.pkl", "type": "model"}]
}
